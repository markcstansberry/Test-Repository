APPLICATION PROGRAMMING INTERFACE (API)

 

ANALYTIC METHODS

 

 

web_analytics_tracker

web_analytics_tracker returns the number of times a URL file or list of URL files was visited. The arguments that can be specified include the time monitoring interval and the zip code of the host from where the file request originated. The url file name argument (urlpage) can be specified as a text string or as an array object.

Signature: web_analytics_tracker(urlpage, timestart, timend, zipcode)

urlpage: The URL file name or URL file array for which site visitor statistics are requested
timestart: The start of the search time monitoring interval
timeend: The end of the search time monitoring interval
zipcode: The zip code of the IP host from which the URL file(s) was originally requested from

Argument values specified with a “*” wildcard character will remove any search constraints for a specific field.

Example Usage

$xvar = web_analytics_tracker(“http://www.clickalot.com/musicplaylist.html”, 20110414.3, 20110414.23, 93215)

This API call to web_analytics_tracker returns the number of visitor page views of the web page URL “http,www.clickalot.com/musicplaylist.html,” that originated from any IP host address with a 93215 United States zip code. The returned results only include web page visits between between 3 A.M on April 14, 2011 and April 14 at 11 P.M. The URL page file name and the number of visitor page views are returned and stored in the key:value array named “xvar.”

Urlpage Argument Values

When an array object is specified as an urlpage argument web_analytics tracker also returns an associative key:value array. The key contains the individual URL files names contiained in the array object. The returned values contain the number of times that each URL file name was viewed.

$xvararray = web_analytics_tracker(google, urlfilesarray, 20110414.3, 20110414.23, 93215)

This API call to web_analytics_tracker returns a key:value array that contains the number of web page visits for each URL address listed in an array named “urlfilesarray.” The contents of $xvararray can be directly displayed in a table or charted with the “tabulate_data” or “chart_data” methods.
